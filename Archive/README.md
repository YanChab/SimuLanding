à l'application Python actuelle (SimuLanding). Ils sont conservés pour mémoire et
traçabilité, mais **ne font plus référence**. Pour la documentation courante, voir
`docs/` (et en particulier `docs/Dossier_de_calcul_complet.md`).

| `Spec.md` | Spécification **historique** du classeur Excel source | Décrit le classeur Excel d'origine, qui n'est plus la référence active. La spécification fonctionnelle vivante est portée par l'app + `docs/`. |
